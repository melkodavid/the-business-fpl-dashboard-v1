# Manager photos

Drop a photo here named `{personKey}.jpg` (e.g. `lu.jpg`, `ibrahim.jpg`) to have
it show up anywhere that manager's avatar appears on the site. `personKey` is
each manager's first name, lowercased — see `data/manager-profiles.json` for
the exact list.

No photo yet for someone? Nothing to do — the site automatically falls back to
a colored initials badge (using their profile color from `manager-profiles.json`)
until a photo file exists.
